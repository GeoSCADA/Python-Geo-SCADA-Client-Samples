# Python Samples
Some simple files showing how to use Python to interact with Geo SCADA using the native Python Client API in Geo SCADA 2025.

Filenames indicate the functions they demonstrate.

Please prepare your Geo SCADA Python environment by following the guide in the 2025 Release Notes or F1 Help.

To use these examples please enter your Geo SCADA login credentials into the code.
